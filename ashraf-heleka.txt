Review questions
    1. What is Semantic HTML? Provides/meaning structure to content
    2. What is HTML used for? HTML provides the core/structure of web content (includes content itself and classification of content)
    3. What is an attribute and where do we put it? Tags that are used to convey content that is not necessarily text based (eg <a>, <img>, etc.)
    4. What is the h1 tag used for? How many times should I use it on a page? H1 is header and is used primarily for the title of a page. Should be used once.
    5. Name two tags that have required attributes <input> and <select>
    6. What do we put in the head of our HTML document? <!DOCTYPE html>
    7. What is an id? Titles that can appear on only 1 element; used to classify and with CSS can be styled.
    8. What elements can I add an id to?  All (except no value in using for elements that only appear once; eg header, footer, body)
    9. How many times can I use the same id on a page? Once
    10. What is a class? Title that can appear on multiple elements;  used to classify and with CSS can be styled.
    11. What elements can I add a class to? All (except no value in using for elements that only appear once; eg header, footer, body)
    12. How many times can I use the same class on a page? Multiple 
    13. How do I get my link to open in a new tab? <a href="" target="_blank">WebName</a>
    14. What is the alt attribute used for? Used with img to explain what image is
    15. How do I reference an id? In CSS #id_Name
    16. What is the difference between a section and a div? A section is used to organize content (provide structure) and could be used to style (via CSS), while a div is exclusively for styling (via CSS).
    17. What is CSS used for? Styling content thats in HTML
    18. How to we select an element? Example - every h2 on the page Use a selector. For h2, in CSS write h2 {
        property: value;
    }
    19. What is the difference between a class and an id? - Give me an example of when I might use each one. Both are similar in that they are used to style. An id can only be used once, while a class may be used many times. For example, class could be used when multiple paragraphs will be styled in the same way and id could be used when one <p> will be distinct from others.
    20. How do we select classes in CSS? .class-Name {}
    21. How do we select a p element with a single class of “human””? .human p {}
    22. What is a parent child selector? When would this be useful? A selector that is able to style parent child relationships (eg elements nested/indented within one another). This is useful when there are many relationships/layers on a webpage
    23. How do you select all links within a div with the class of sidebar? .sidebar div a {}
    24. What is a pseudo selector?  Selector that styles a special state of an element (e.g., hover)
    25. What do we use the change the spacing between lines? line-height property
    26. What do we use to change the spacing between letters? letter-spacing property
    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
    28. How do I add a 1px border around my div that is dotted and black? div {
        border: 1px dotted black;
    }
    29. How do I select everything on the page? * {}
    30. How do I write a comment in CSS? /*Comment */
    31. How do I find out what file I am in, when I am using the command line? pwd
    32. Using the command line - how do I see a list of files/folders in my current folder? ls (ls -la to see hidden files)
    33. How do I remove a file via the command line? Why do I have to be careful with this? rm followed by filename. This permanently deletes the file (no recycling bin restore)
    34. Why should I use version control? To track changes to content and to allow for multiple people to work on same content simultaneously
    35. How often should I commit to github? Commit often (several times per hour if creating lots of content), publish once
    36. What is the command we would use to push our repo up to github? git push

Stretch Questions

    1. What is the difference between an inline element and a block element? Inline takes does not start on a new line, while block element does. Inline takes only enough space for content, while block takes up entire width available.
    2. What happens when an element is positioned absolutely? It is positioned relative to its nearest positioned ancestor.
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? Use diplay options (flex)
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default. Block <h1>-<h6>, <div>, <p>; Inline: <a>, <span>, <em>; Inline-block: <img>
    5. In your own words, explain the box model. What is the fix for the box model?  A model fo positioning HTML elements by using CSS. All shapes are boxes and all elements correspond to pixels.
